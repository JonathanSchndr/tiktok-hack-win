4. Die Teilnahme am Contest erfolgt bei TikTok durch das Lösen der verschiedenen Rätsel. Jedes Rätsel gibt ein Teil des Amazon-Gutschein-Codes frei. Alle 3 Teile zusammengefügt ergeben den Gutscheincode.
